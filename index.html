Help & FAQ

Use Single to analyze one text or Batch CSV/Excel to label many rows.

  • CSV/Excel format: Include a column with the text to analyze.
  • Text column: After upload, select the column name that contains the text.
  • Language: Choose Filipino or English to match your input language.
